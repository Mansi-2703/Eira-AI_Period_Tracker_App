# Eira

HerCycle (Eira) is a Flutter-based cycle tracker that pairs smooth authentication, journaling, and profile widgets with a secure REST backend to store quizzes, predictions, and logs. Bonus: the backend feeds an LSTM-powered ML pipeline so the app can predict your next cycle length and keep the calendar aligned with your latest data.
